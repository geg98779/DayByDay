## 什么时候使用绝对定位？
- 元素出现在一个天马行空的位置
- 元素是否存在，不影响其他元素的排列
- 单个元素在某个区域内水平垂直居中

## fixed和absolute的区别

fixed是绝对定位的一种特殊情况，它们的参考系不一样
- absolute参考有定位的父元素
- fixed参考视口

## vite处理svg矢量图
```bash
# 安装插件
npm i --save-dev vite-plugin-svg-icons@2.0.1
# vite.config.js 配置
plugins: [
    vue(),
    createSvgIconsPlugin({
        # 指定需要缓存的图标文件夹
        iconDirs: [path.resolve(process.cwd(), 'src/assets/icons')],
        symbolId: 'icon-[name]'
    )
],
# main.js 注册svg插件
import 'virtual:svg-icons-register'
```
