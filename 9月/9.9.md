## handcoding debounce & throttle
```js
function debounce(fn,wait){
    let timer = null
    return function(){
        // let context = this
        // let args = arguments
        if(timer){
            clearTimeout(timer)
            timer = null
        }
        timer = setTimeout(() => {
            fn.apply(this,arguments)
        },wait)
    }
}
```
直接这样写可以吗? 为什么不可以? 为啥一定要用一个变量来保存this和命令行参数
this 和 arguments 的值在 setTimeout 的回调函数中会丢失
所以需要两个闭包变量来一起打包出去 注意这里timer变量也是闭包变量 这里有两层闭包 使后续的访问正常

纠正：
```js
function debounce(fn,wait){
    let timer = null
    return function(){
        let context = this
        let args = arguments
        if(timer){
            clearTimeout(timer)
            timer = null
        }
        timer = setTimeout(() => {
            fn.apply(context, args)
        },wait)
    }
}
```
