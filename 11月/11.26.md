## jsonp
**仅支持GET请求**

- 前端准备
参数构成：
callback：是 JSONP 约定的 “回调函数名参数名”（后端会识别这个参数，获取前端定义的回调函数名）；
handleRes：是前端实际定义的 全局回调函数名（前端会提前在 window 上挂载 window.handleRes = function(data) { ... }，用于接收后端返回的跨域数据）。
- 动态创建script标签
- 后端响应
- 前端执行

```js
function addScript(src) {
    const script = document.createElement("script")
    script.src = src
    script.type = "text/javascript"
    document.body.appendChild(script)
}

addScript("http://xxx.xxx.com/xxx.js?callback=handleRes")

function handleRes(res) {
    console.log(res)
}
```
